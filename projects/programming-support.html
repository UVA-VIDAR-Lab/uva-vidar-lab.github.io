Support for Visualization Programming


Programming based tools are some of the most popular and expressive tools for creating data visualizations. Yet programmatically creating visualization designs is often a cumbersome process, requiring technical expertise that not all visualization designers have. There is a need to understand how we can help users leverage the expressive power of programming tools like D3 while allowing them to have creative control over their designs. We investigate how people write and prototype data visualizations through code, with a focus on identifying common patterns and streamlining workflows. This includes analyzing real-world D3 programs, understanding user coding behaviors, and building tools that better align with designersâ€™ needs.
